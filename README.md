# AlexTextProcessing

- 用wxpython写的简单的文本处理的界面工具

## 环境

 - python3.6 + 相关库

## 功能
  
  - ｋｅｙ ｗｏｒｄｓ　
     + 给定一句话，得到这句话的关键词
     + 需要读入存储word2vec的４个文件
     + 可选择显示的词的个数，点击ｓａｖｅ可存储ｔｘｔ到本地
  - relatives
     + 给定一句话，得到这句话的相关词
     + 可选择显示的词的个数，点击ｓａｖｅ可存储ｔｘｔ到本地
  - asps
     + 给定一句话，得到这句话的核心词
     + 需要读入ｗｏｒｄｓ文档，以文档内的词对句子进行搜索，ｍｏｄｅｌ文件夹内有模板
     + 可选择显示的词的个数，点击ｓａｖｅ可存储ｔｘｔ到本地
     
## 使用

 - 将训练好的word2vec的4个文件放入model文件夹内
 - 将写好的ｗｏｒｄｓ放在ｍｏｄｅｌ文件夹内
 - 修改main.py中MyModel的参数为word2wec的model文件的路径
 - 用python运行main.py
